  安装步骤

  1. 浏览器加载扩展：打开 chrome://extensions → 开启开发者模式 → 「加载已解压的扩展程序」→ 选择 extension/ 目录
  2. 记下扩展 ID：加载后页面上会显示 32 位 ID
  3. 安装本地组件：右键以管理员身份运行 native-host/install.bat，输入扩展 ID
  4. 刷新测试：访问目标视频页面，点击扩展图标查看嗅探结果
